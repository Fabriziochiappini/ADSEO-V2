# ADSEO2 - SEO Automation Platform

ADSEO2 is a high-performance SEO automation platform built with Next.js 15, Gemini 1.5 Flash, and DataForSEO. It allows users to generate multi-domain SEO network strategies with visual relationship mapping.

## üöÄ Features

- **Topic Analysis**: Real-time keyword ideas and search volume via DataForSEO.
- **AI Strategy**: Deep niche analysis and site blueprinting using Gemini 1.5 Flash.
- **Network Visualization**: Animated graph showing site relationships.
- **Modern UI**: Dark-themed, responsive dashboard built with Tailwind CSS and Framer Motion.

1. **Clone the repository**:
   ```bash
   git clone https://github.com/Fabriziochiappini/ADSEO-V2.git
   cd ADSEO-V2
   ```

2. **Install dependencies**:
   ```bash
   npm install
   ```

3. **Configure Environment Variables**:
   Create a `.env.local` file in the root directory and add:
   ```env
   GEMINI_API_KEY=your_key
   DATAFORSEO_USERNAME=your_username
   DATAFORSEO_PASSWORD=your_password
   ```

4. **Run the development server**:
   ```bash
   npm run dev
   ```
